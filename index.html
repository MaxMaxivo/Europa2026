Cambios de destino
Decisiones abiertas
Alta prioridad
¿Mantener Marsella o mover noches?
Criterio sugerido: si el traslado complica o encarece, sumar una noche a Niza, Florencia, Roma o Amalfi puede mejorar el ritmo.
Camino
Ejecutar Madrid → Sarria sin apuro
Decisión ejecutada: pasajes Renfe comprados para el 02 Ago. Falta preparar la ejecución fina: Cercanías T4 → Chamartín, margen con equipaje, comida/descanso en estación y control del transbordo de 13 min.
Camino
5 etapas o 6 etapas hasta Santiago
Plan base: 03 Ago Sarria → Portomarín, 04 Ago Portomarín → Palas de Rei, 05 Ago Palas de Rei → Arzúa, 06 Ago Arzúa → O Pedrouzo, 07 Ago O Pedrouzo → Santiago. El tramo crítico es Palas de Rei → Arzúa, 29 km; si pesa demasiado, agregar una noche intermedia y llegar a Santiago el 08 Ago.
Costa Azul
Base en Niza vs. dividir Niza / Marsella
Niza funciona mejor como base liviana para Mónaco, Èze, Antibes o Cannes; Marsella suma si interesa Calanques y puerto viejo.
Final del viaje
Volver a Madrid desde Nápoles o Roma
No reservar la última noche hasta confirmar vuelo interno. Prioridad: llegar a Madrid con margen para PU501.
